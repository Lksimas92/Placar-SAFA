MANUAL DE INSTALAÇÃO

Placar Esportivo SAFA

Guia Completo para Instalação Offline

Versão 1.0 - 2026

Este manual descreve o processo completo de instalação e configuração do Placar Esportivo SAFA para uso offline em notebooks.

1. REQUISITOS ANTES DE COMEÇAR

Você vai precisar de:

  • Um notebook com Windows, Mac ou Linux
  • Acesso à Internet (durante a preparação)
  • Um pendrive com pelo menos 200MB de espaço
  • Tempo: aproximadamente 15-20 minutos para preparação

2. PASSO 1: BAIXAR O PROJETO (COM INTERNET)

Opção A: Pelo Bolt (Recomendado)

  1. Acesse o Bolt onde o projeto está hospedado
  2. Clique no botão "Download" ou "Export"
  3. Escolha "ZIP" como formato
  4. Salve o arquivo em seus Downloads

Opção B: Pelo Git

  1. Abra o Terminal (CMD ou PowerShell)
  2. Digite o comando abaixo:
git clone <url-do-repositorio>
  1. Pressione Enter e aguarde

3. PASSO 2: INSTALAR NODE.JS (COM INTERNET)

O Node.js é necessário para rodar o projeto. Instale uma única vez:

  1. Acesse: https://nodejs.org
  2. Clique em "LTS" (versão recomendada)
  3. Clique em "Download"
  4. Execute o instalador
  5. Clique em "Next" em todas as telas
  6. Na última tela, clique em "Finish"

Para verificar se instalou corretamente:

  1. Abra o Terminal (CMD ou PowerShell)
  2. Digite:
node --version

Se aparecer um número (ex: v18.0.0), funcionou!

4. PASSO 3: INSTALAR DEPENDÊNCIAS (COM INTERNET)

Após descompactar o projeto:

  1. Abra o Terminal (CMD ou PowerShell)
  2. Navegue até a pasta do projeto:
cd C:\Users\SeuUsuario\Downloads\project

(ou onde você salvou)

  1. Digite o comando:
npm install

Pressione Enter e aguarde (pode levar 2-5 minutos)

Após terminar, digite:

npm run build

Aguarde novamente (2-3 minutos)

5. PASSO 4: PREPARAR PARA OFFLINE

Agora você tem a pasta "dist" pronta para rodar offline:

  1. Localize a pasta: C:\Users\SeuUsuario\Downloads\project\dist
  2. Copie TODA a pasta "dist" para um pendrive
  3. Você pode deletar o resto do projeto depois
Resumo: Apenas a pasta "dist" é necessária!

6. PASSO 5: USAR NO GINÁSIO (SEM INTERNET)

Você tem 2 opções para rodar o placar offline:

OPÇÃO A: Com Node.js (Recomendado)

  1. Conecte o pendrive ao notebook
  2. Copie a pasta "dist" de lá para o notebook
  3. Abra o Terminal (CMD ou PowerShell)
  4. Digite:
npx serve C:\caminho\da\pasta\dist
  1. Abra o navegador em: http://localhost:3000

OPÇÃO B: Com Python

  1. Instale Python: https://python.org (COM INTERNET)
  2. Copie a pasta "dist" do pendrive
  3. Abra o Terminal (CMD ou PowerShell)
  4. Navegue até a pasta:
cd C:\caminho\da\pasta\dist
  1. Digite:
python -m http.server 8080
  1. Abra o navegador em: http://localhost:8080

7. USANDO O PLACAR

Depois de abrir no navegador:

  1. Escolha o esporte (Futsal, Vôlei, Basquete, Handebol)
  2. Configure a duração dos períodos
  3. Clique em "Começar Jogo"
  4. Use os botões + e - para alterar o placar
  5. Clique nos nomes para personalizar os times
  6. Clique nas imagens para adicionar logos
IMPORTANTE: Todos os dados são salvos localmente no notebook!

8. TROUBLESHOOTING (Resolvendo Problemas)

Problema: "Comando npx não encontrado"

Node.js não foi instalado. Instale novamente a partir de https://nodejs.org

Problema: "Conexão recusada em localhost:3000"

Execute novamente o comando npx serve

Problema: "Pasta dist não encontrada"

Execute npm run build novamente

Problema: "Não consigo editar os nomes dos times"

Clique no nome do time (não na imagem) para editar

9. RESUMO RÁPIDO

COM INTERNET (antes de ir ao ginásio):

  1. Instale Node.js (uma única vez)
  2. npm install
  3. npm run build
  4. Copie a pasta "dist" para um pendrive

SEM INTERNET (no ginásio):

  1. Copie "dist" do pendrive
  2. npx serve ./dist
  3. Abra http://localhost:3000 no navegador
  4. Use normalmente - tudo é salvo localmente!

10. TABELA: O QUE PRECISA DE INTERNET

Ação Precisa de Internet?
Baixar o projeto ✓ SIM
Instalar Node.js ✓ SIM
npm install ✓ SIM
npm run build ✗ NÃO
Copiar "dist" para pendrive ✗ NÃO
Rodar com npx serve ✗ NÃO
Usar o placar no ginásio ✗ NÃO

11. DICAS IMPORTANTES

  • Antes de ir ao ginásio: Teste o app em casa executando npx serve ./dist
  • Verifique: Se o navegador abre corretamente em http://localhost:3000
  • Pratique: Os comandos básicos e a interface do placar
  • No ginásio: Tenha um segundo notebook como backup
  • Dados: Todos os jogos são salvos automaticamente no navegador
  • Recuperação: Para recuperar um jogo salvo, procure na tela inicial em "Jogos Salvos Localmente"

SUPORTE

Se tiver dúvidas durante a instalação, revise os passos acima ou peça ajuda a um responsável técnico.

O Placar Esportivo SAFA funciona 100% offline após a instalação inicial.
